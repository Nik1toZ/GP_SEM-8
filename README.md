# Bridge Runner

3D-раннер на Unity 6 с процедурно генерируемой трассой. Игрок автоматически бежит вперёд по узкой дорожке с пропастями и препятствиями, строит мосты в реальном времени, стреляя блоками трёх разных типов.

## Техническое задание

3D раннер: Персонаж бежит вперёд сам, на пути появляются пропасти. Игрок стреляет блоками и строит мосты прямо на ходу. Блоков мало, поэтому старые нужно переиспользовать.

**Ключевые механики:**

1. **Стрельба блоками** — игрок целится и стреляет платформами, которые летят по дуге и приклеиваются к поверхности. Криво упавший блок через несколько секунд ломается.
2. **Три вида блоков** — обычный, пружинный и наклонный.
3. **Возврат блоков** — всего несколько штук, но старые блоки позади игрока сами возвращаются в запас.

## Что реализовано

- Процедурная генерация трассы из 9 видов сегментов (прямые, пропасти разной ширины, узкие участки, стены, смещения, параллельные дорожки).
- Три типа блоков: **Flat** (обычный мост), **Bouncy** (пружинный, подбрасывает игрока), **Ramp** (рампа для взбегания на стены).
- Прицеливание с предпросмотром траектории и маркером точки приземления.
- Инвентарь по 3 блока каждого типа с автовозвратом старых блоков и безвозвратной потерей упавших в пропасть.
- Game Over по падению в пропасть и по застреванию у препятствия.
- Сохранение лучшего результата между запусками через `PlayerPrefs`.
- Главное меню с правилами игры, экран Game Over с рестартом.
- Небо, анимированный персонаж-свинка, динамические тени.

## Управление

| Клавиша | Действие |
|---|---|
| `1` / `2` / `3` | Переключение типа блока (обычный / пружинный / наклонный) |
| `ЛКМ` | Выстрел блока |
| `A` / `D` | Движение в стороны |
| `Пробел` | Прыжок |
| `T` | Показать / скрыть траекторию полёта блока |
